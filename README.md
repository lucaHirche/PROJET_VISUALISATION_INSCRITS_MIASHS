# PROJET_VISUALISATION_INSCRITS_MIASHS

Ce projet a été réalisé dans le cadre du **Défi 2** du cours de Projet Info (L2 MIASHS, Semestre 4). Il consiste en une interface web permettant la visualisation et l'analyse des effectifs inscrits en **L2 MIASHS à l'Université Toulouse - Jean Jaurès (UT2J)**.

L'objectif principal est de présenter des données statistiques sous forme de graphiques interactifs pour faciliter la comparaison et l'analyse des effectifs.

## 🚀 Technologies utilisées

- **HTML5** : Structure des pages web.
- **CSS3** : Mise en forme et design personnalisé (utilisation de polices comme *Montserrat*, arrière-plans dynamiques).
- **Chart.js** : Bibliothèque JavaScript utilisée pour la génération des graphiques de visualisation de données.

## 📂 Structure du projet

Le projet est organisé de la manière suivante :

- `index.html` : Page d'accueil présentant le projet et l'équipe.
- `src/pages/` : Contient les différentes pages de visualisation (`comparaison1.html` à `comparaison5.html`, `defi-2.2.html`, `defi-2.3.html`).
- `src/styles/` : Feuilles de style CSS organisées par module et page.
- `src/assets/` : Ressources graphiques (images et icônes).

## 🛠️ Installation et Utilisation

Comme il s'agit d'un projet web statique, aucune installation complexe n'est requise.

1. Clonez le dépôt ou téléchargez les fichiers.
2. Ouvrez le fichier `index.html` dans votre navigateur web préféré pour accéder à l'interface d'accueil.
3. Naviguez à travers les différentes pages de comparaison via les liens fournis (si présents dans l'interface).
